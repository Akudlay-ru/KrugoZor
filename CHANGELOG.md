# История изменений

## 11.5 — 2025-08-25

- Обновлена версия приложения до `11.5`.
- Сохранена логика минималистичного окна поверх всех окон.
- Сохранены функции crop, зеркалирования, хромакея, трея и виртуальной камеры.

## 11.4 — 2025-08-25

- Команда «Выключить камеру» дополнительно скрывает окно через синхронизацию с чекбоксом трея.
- При показе окна из трея не поднимается `Qt.Window`; кнопка в панели задач не появляется, окно остаётся `Qt.Tool`.
- В виртуальной камере режимы «По ширине окна» и `Letterbox` используют квадратный ROI из окна без растяжений.

## До 11.4

- Плавающее окно веб-камеры поверх всех окон.
- Круглая и квадратная форма окна.
- Crop области кадра.
- Настройка прозрачности.
- Хромакей RGB/HSV.
- Зеркалирование окна и виртуальной камеры.
- Работа из системного трея.
- Сохранение настроек в `config.json`.
- Интеграция с калькулятором Windows через NumLock.
