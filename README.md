# КругоЗор / KrugoZor

КругоЗор — минималистичная Windows-утилита для видеозвонков, презентаций и стримов.

Программа выводит изображение с веб-камеры в небольшое плавающее окно поверх остальных окон. Окно можно использовать как визуальное присутствие при демонстрации экрана. Дополнительно поддерживается отправка изображения в виртуальную камеру для Zoom, Teams, OBS и других приложений.

## Возможности

- плавающее окно поверх всех окон;
- форма окна: круг или квадрат;
- выбор области кадра — crop;
- зеркалирование окна и виртуальной камеры;
- настройка прозрачности окна;
- хромакей с RGB/HSV-настройками;
- работа через системный трей;
- сохранение пользовательских настроек в `config.json`;
- логирование в пользовательскую папку приложения;
- управление калькулятором Windows через NumLock;
- подготовка к сборке в `.exe` через PyInstaller.

## Требования

- Windows 10/11;
- Python 3.10+;
- рабочая веб-камера;
- для виртуальной камеры: установленный драйвер виртуальной камеры, например OBS Virtual Camera.

## Установка из исходников

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python KruGoZor_11_5.py
```

Если виртуальная камера не нужна, программа должна запускаться без `pyvirtualcam`, но соответствующая функция будет недоступна.

## Пользовательские данные

Приложение хранит настройки и служебные файлы вне папки репозитория:

```text
%APPDATA%\KrugoZor\config.json
%APPDATA%\KrugoZor\Logs\krugozor.log
%APPDATA%\KrugoZor\Snapshots\
```

Эти файлы не должны попадать в GitHub.

## Сборка exe

Подробная инструкция вынесена в [`BUILD.md`](BUILD.md).

Базовая команда:

```bash
pyinstaller --noconsole --onefile --name KrugoZor KruGoZor_11_5.py
```

Если в репозитории есть `icon.ico`, можно добавить иконку:

```bash
pyinstaller --noconsole --onefile --name KrugoZor --icon icon.ico KruGoZor_11_5.py
```
