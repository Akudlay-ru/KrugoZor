# KrugoZor

КругоЗор — Windows-утилита «пузырь-камера»: окно с веб-камерой поверх всех окон, хромакей, кроп, мягкая виньетка и виртуальная камера для Zoom/Teams/OBS/Телемоста.

## Текущая версия

Актуальный рабочий файл: `KruGoZor.pyw`.

Старая опубликованная линия `KruGoZor_11_5.py` больше не является основной. Старые файлы должны оставаться только в истории Git, а не в корне релизной ветки.

## Что входит в текущую сборку

- плавающее окно камеры: круг / квадрат;
- работа поверх всех окон и режим «прокликиваемого» окна;
- кроп исходного кадра;
- динамический кроп по лицу;
- FaceDirector через `kgz_face_director.py`;
- MediaPipe FaceMesh при наличии пакета `mediapipe`;
- fallback на OpenCV Haar при отсутствии MediaPipe;
- ручной, фиксированный и автоматический масштаб динамического кропа;
- хромакей с несколькими пипетками и HSV-режимом;
- мягкая виньетка;
- виртуальная камера через `pyvirtualcam`;
- режим «лицо в круг» для видеосервисов;
- PTZ-контур через OpenCV CAP_PROP_PAN / TILT / ZOOM;
- горячие клавиши;
- работа из трея;
- внешний стиль `style.qss` рядом с программой;
- сохранение пользовательских настроек в `config.json`.

## Быстрый запуск

```bat
py -m pip install -r requirements.txt
py KruGoZor.pyw
```

Опционально для более точного трекинга лица:

```bat
py -m pip install -r requirements-optional.txt
```

Если `mediapipe` не установлен, программа не должна падать: FaceDirector откатывается на Haar/fallback-логику.

## Сборка EXE

```bat
build.bat
```

Скрипт собирает `KruGoZor.pyw` через PyInstaller в `dist\KruGoZor.exe` и добавляет `style.qss` в комплект.

## Важные ограничения

- Целевая среда: Windows 10/11.
- PTZ зависит от камеры и драйвера. OpenCV/DirectShow не всегда отдаёт реальные Pan/Tilt/Zoom.
- Виртуальная камера требует установленный backend, совместимый с `pyvirtualcam`.
- Runtime-проверка с физической камерой и виртуальной камерой выполняется только на Windows-машине пользователя.
